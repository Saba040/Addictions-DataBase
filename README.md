ეს პროგრამა წარმოადგენს მონაცემთა შეგროვებისა და ვიზუალიზაციის ხელსაწყოს, რომელიც აგროვებს ადამიანების დამოკიდებულებებთან დაკავშირებულ ინფორმაციას (სიგარეტი, ალკოჰოლი და ჯანმრთელობა) და ინახავს მას SQLite ბაზაში. მომხმარებელი შეყავს მონაცემები (ასაკი, სქესი, ქორწინების სტატუსი, კვება, BMI და სხვ.), რომელთა დამატება, განახლება და წაშლა ხდება ბაზაში ID-ის მიხედვით.

შემდეგ ეტაპზე პროგრამა ახორციელებს ანალიტიკას და ვიზუალიზაციას სქესის, ასაკობრივი ჯგუფების და ქორწინების სტატუსის მიხედვით. დიაგრამების საშუალებით ვიხილავთ, როგორია დამოკიდებულების სტატისტიკა სხვადასხვა კატეგორიებში (მაგ. დაქვრივებული მამაკაცების სიგარეტის მოხმარება). პროექტი იყენებს matplotlib და numpy ბიბლიოთეკებს სურათების შესაქმნელად და sqlite3 მოდულს მონაცემთა ბაზასთან დასაკავშირებლად.
